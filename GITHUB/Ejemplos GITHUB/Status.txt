Comandos: 
    git status


Funciones del "git status" Este comando muestra el estado actual del repositorio, indicando si hay cambios sin confirmar
(sin commitear), si los archivos están en el área de preparación (staged), 
si están sin seguimiento, entre otros. Es útil para ver de manera rápida qué archivos han cambiado, cuáles están listos para ser enviados al repositorio remoto y cuáles aún necesitan ser procesados.



Este comando te permitirá ver si tienes cambios pendientes de hacer commit, si necesitas usar git add para agregar archivos al área de preparación, o si hay archivos sin seguimiento que necesitan ser añadidos al control de versiones. En resumen, git status es una herramienta esencial para conocer el estado actual de tu repositorio y tomar las acciones correspondientes.
